# SenticScraper

This project extracts articles from URLs and performs text analysis as per the assignment.

## Files
- `scraper_analyzer.py` : Main Python script.
- `requirements.txt` : Python dependencies.
- `README.md` : Instructions.

## Usage
1. Place `Input.xlsx`, `Output Data Structure.xlsx`, the `StopWords/` folder, and the `MasterDictionary/` folder in this directory.
2. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the script:
   ```bash
   python scraper_analyzer.py --input Input.xlsx --output Output.xlsx --articles_dir articles --output_structure "Output Data Structure.xlsx"
   ```
4. The script will:
   - Save each article as `<URL_ID>.txt` inside `articles/`.
   - Produce `Output.xlsx` with computed variables in the exact order required.
